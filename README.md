## 目录结构

- 01.Scenes  
    场景文件与有关附属文件
- 02.Scripts  
    根据功能块再分
- 03.Prefabs  
    根据种类再分
- 04.Images  
    根据种类再分、人物、道具、设备等
- 05.Audios  
    01.Musics  
    02.SFXs
- 06.Animations  
    根据对象再分层次
- 07.UI   //专门存放UI资源
- 08.FX      //专门存放特效（粒子系统等）
- 09.Fonts
- 10.Resources  //用于动态加载资源（尽量不用）
